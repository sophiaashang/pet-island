# 萌宠任务岛 - 产品规格文档

## 1. 概念与愿景

一款针对 5-12 岁儿童的奖励机制任务管理 + 宠物养成游戏。两个孩子（5岁元元、11岁新北）各拥有自己的宠物岛屿，通过完成每日任务获得金币，养宠物、买衣服、抽新宠。两个孩子的宠物之间有羁绊关系——一方完成任务，另一方的宠物也会获得"友情能量"。

核心感受：**可爱、干净、有成就感**。不是任务管理器，是一个小世界。

---

## 2. 产品架构

### 2.1 角色体系

| 孩子 | 年龄 | 界面风格 | 任务数量 | 奖励节奏 |
|------|------|---------|---------|---------|
| 元元 | 5岁，大班下学期 | 大图+大按钮，几乎无文字，图标为主 | 每日5-7项 | 每完成1个即正反馈 |
| 新北 | 11岁，5年级下学期 | 青少年简约风，文字为主 | 每日8-12项，含日+周混合 | 累积型，需要积累才有爽感 |

### 2.2 页面结构

```
启动页（选孩子）
├── 元元岛屿（5岁版）
│   ├── 今日任务（图标+进度）

│   ├── 抽新宠（扭蛋机）
│   └── 图鉴（收集展示）
│
├── 新北岛屿（11岁版）
│   ├── 今日任务列表
│   ├── 周目标/里程碑
│   ├── 我的宠物
│   ├── 宠物背包
│   ├── 抽新宠
│   └── 图鉴

    └── 任务模板管理（增加/删除/修改任务）
```

## 3. 宠物系统

### 3.1 初始宠物

| 孩子 | 宠物名 | 形象 | 性格 | 进化阶段 |
|------|--------|------|------|---------|
| 元元 | 小黄鸭（DUCKY） | 圆润黄色小鸭，像素风 | 活泼话多，喜欢被夸 | 3阶段：DUCKY → 胖鸭 → 天鹅鸭 |
| 新北 | 小灰狼（WOLFY） | 银色毛小狼，像素风 | 酷酷的，话少但很忠诚 | 3阶段：WOLFY → 银狼 → 雪狼王 |

### 3.2 可收集宠物（扭蛋获取）

| 宠物名 | 形象 | 稀有度 | 性格小设定 |
|--------|------|--------|-----------|
| 棉花羊 | 白色绵羊，云朵身体 | ⭐⭐ 普通 | 整天睡觉，叫声像棉花糖一样软 |
| 磷磷龟 | 绿色小龟，背上有花园 | ⭐⭐ 普通 | 爬得很慢，但从不放弃 |
| 烈焰鸡 | 红色公鸡，像素羽毛着火 | ⭐⭐⭐ 稀有 | 每天早上打鸣，骄傲得很 |
| 泡泡龙 | 蓝色小龙，吹出的泡泡会发光 | ⭐⭐⭐ 稀有 | 泡泡里藏着它偷偷许的愿望 |
| 月光狐 | 银白色狐狸，尾巴像极光 | ⭐⭐⭐⭐ 珍稀 | 只在深夜出现，喜欢听秘密 |
| 星际熊 | 紫色小熊，肚子上有星星 | ⭐⭐⭐⭐ 珍稀 | 传说它来自银河尽头 |

### 3.3 宠物状态

- **饥饿值**（0-100）：随时间下降，喂食可恢复
- **心情值**（0-100）：完成任务↑，连续不完成↓，长时间不登录↓至忧郁状态
- **友情值**：与对方孩子的宠物的羁绊，完成任务时双方同时+1

### 3.4 宠物互动

- 点击宠物：宠物播放动画 + 说话（3种随机台词）
- 喂食：消耗食物道具，饥饿值恢复，开心+5
- 陪玩：消耗玩具，开心+10
- 换装：消耗金币/衣服配件，宠物外观变化

### 3.5 进化机制

宠物每个阶段有3个外观形态，达到一定等级时自动进化。等级通过"完成的任务次数"累积。

## 4. 任务系统

### 4.1 任务字段结构

```typescript
interface Task {
  id: string;
  name: string;                    // 显示名称
  icon: string;                    // emoji图标
  type: 'daily' | 'weekly';       // 每日/每周
  targetCount: number;              // 目标次数（如跳绳200次=1次完成）
  currentCount: number;            // 当前完成进度
  coinReward: number;              // 完成奖励金币
  forChild: 'yuanyuan' | 'xinbei';
  category: 'study' | 'sport' | 'life' | 'piano';
  isCountable: boolean;            // 是否是计数任务（如跳绳）
  countableTarget?: number;        // 计数目标（如跳绳200次）
  milestones?: MilestoneTask[];    // 里程碑任务（如PET、高考钢琴10级）
}
```

### 4.2 预置任务列表

**元元（5岁）每日任务：**
- 🟢 英语作业（小i英语/绘本）
- 🟢 英语小书跟读
- 🟢 高斯数学每日一练
- 🟢 钢琴练习（哈农/小汤3，每首曲子5次）
- 🟢 跳绳200次
- 🟡 随机生活任务（每天从库中抽取2-3个）：洗袜子、叠衣服、自己睡觉、自己洗漱、扎马步

**新北（11岁）每日任务：**
- 🟢 校内语数英作业
- 🟢 英语作业
- 🟢 高斯数学每日一练
- 🔵 周四：高斯数学作业
- 🔵 周五：PET阅读2页
- 🔵 预复习（最新进度）
- 🟢 钢琴练习1小时（央音10级）
- 🟢 体育运动
- 🟡 钢琴任务：可计数，每练习1次计一次（目标60次=1小时）

**新北里程碑任务（长期目标）：**
- 🎯 PET备考：累计完成50天打卡 → 大奖励
- 🎯 央音钢琴10级：累计练习300小时 → 解锁"雪狼王"进化
- 🎯 期末考试全优：各科90分以上 → 宠物全套新衣服

### 4.3 任务完成流程

1. 孩子点"完成任务"按钮
2. 对于计数任务（如跳绳、钢琴练习）：孩子或家长输入完成次数（计数器界面）
3. 系统记录完成，发放金币
4. 两人的宠物同时获得心情+1
5. 任务列表刷新进度

### 4.4 奖励机制

| 完成方式 | 金币奖励 |
|---------|---------|
| 每日普通任务 | +10~20金币 |
| 计数任务（跳绳/钢琴）| +5金币/次 |
| 全部任务完成（日结）| +50额外奖励 |
| 连续7天全完成 | 宠物食物礼包 |
| 里程碑达成 | 100~500金币 + 扭蛋券 |

## 5. 扭蛋/抽宠系统

### 5.1 扭蛋机

- 单抽：80金币/次
- 十连抽：720金币/次（优惠10%）
- 扭蛋券：可免费单抽1次

### 5.2 扭蛋结果分布

- 棉花羊：30%
- 磷磷龟：30%
- 烈焰鸡：20%
- 泡泡龙：15%
- 月光狐：4%
- 星际熊：1%

### 5.3 保底机制

连续30抽未出珍稀（⭐⭐⭐⭐），第31抽必出月光狐或星际熊之一。

## 7. 视觉风格

- **整体风格**：像素风 + 柔和马卡龙色，颜色干净不乱
- **字体**：圆体，中文无衬线
- **图标**：emoji为主，辅助少量自定义像素图标
- **主色**：
  - 元元岛屿：明黄 + 草绿（活力感）
  - 新北岛屿：银灰 + 深蓝（少年感）
- **宠物**：像素风 GIF/动画，16x16基础尺寸放大显示

## 8. 技术方案

- **框架**：React + TypeScript + Vite
- **样式**：TailwindCSS（简洁、响应式）
- **状态**：React Hooks + localStorage（持久化，无后端）
- **部署**：直接 deploy 到 web server，生成 URL 分享
- **数据存储**：每个孩子的数据在 localStorage 里，key = `pet-island-{childId}`

### 核心数据结构

```typescript
interface GameState {
  currentChild: 'yuanyuan' | 'xinbei';
  children: {
    yuanyuan: ChildProfile;
    xinbei: ChildProfile;
  }
}

interface ChildProfile {
  name: string;
  pet: Pet;
  tasks: Task[];
  inventory: Item[];
  wardrobe: ClothingItem[];
  petsCollection: Pet[];   // 已收集的宠物（除初始宠外）
  unlockedCostumes: string[];
  totalCoins: number;
  stats: {
    totalTasksCompleted: number;
    totalCoinsEarned: number;
    loginDays: number;
    lastLoginDate: string;
    currentStreak: number;  // 连续登录天数
  };
  milestones: {
    [key: string]: { progress: number; completed: boolean; rewardsClaimed: boolean };
  };
}
```
